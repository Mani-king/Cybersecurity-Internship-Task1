# Cybersecurity Internship - Task 1

## Foundation & Environment Setup

## Overview

This repository contains the work completed for Task 1 of the Cybersecurity & Ethical Hacking Internship Program. The objective of this task was to build a strong foundation in cybersecurity concepts, networking, cryptography, Linux fundamentals, and security tools while setting up a professional cybersecurity lab environment.

---

## Objectives

* Understand the CIA Triad (Confidentiality, Integrity, Availability)
* Learn common cyber threats and attack vectors
* Set up a cybersecurity lab using VMware and Kali Linux
* Gain hands-on experience with Linux commands
* Learn networking fundamentals and cryptography concepts
* Familiarize with cybersecurity tools such as Nmap, Wireshark, Burp Suite, and Netcat
